# 타임라인 편집기 — 자막·글자·효과·효과음·음악 선택

날짜: 2026-07-06 · 상태: 승인됨 (사용자)

## 왜

자동 산출물에서 사용자가 고를 수 있는 게 거의 없다. BGM은 LLM 무드 자동 선곡뿐이고(무드당 1곡), 효과음은 기능 자체가 없다. 사용자 요청: "VLLO처럼 편집하면서 선택 — 라이브러리가 많은 게 아니라 내가 원하는 걸 선택이라도 할 수 있으면 좋겠어."

## 결정 사항 (사용자 확인)

- 선택 시점: **결과 화면에서** (업로드 폼은 지금처럼 자동)
- 선택 단위: **자막 스타일·효과·음악은 영상 전체 1개, 글자·효과음은 구간별**
- SFX 라이브러리: **기본 세트 포함** (CC0 10개 내외)
- BGM 라이브러리 확충: 이번엔 안 함 (선택 UI만)
- 미리보기: 실시간 합성은 불가 — 스타일 데모 클립(사전 렌더) + 썸네일 자막 오버레이 근사 + 음원 미리듣기. 실물 반영은 "다시 만들기"(분석 재사용 rebuild)

## 화면 구조 (결과 화면, 기존 교정 패널 대체)

```
┌ 전체 설정 바 ────────────────────────────┐
│ 자막 스타일 ▾ (▶ 스타일 데모)  효과: 점프컷☑ 펀치인☑ │
│ 음악: 자동 선곡 / 끄기 / 곡별 라디오 행 (▶ 미리듣기)  │
├ 타임라인 (가로 스크롤, 구간 폭 ∝ 길이) ────┤
│ 영상   [썸네일1][썸네일2][썸네일3]…        │
│ 글자   [자막 문구][문구][문구]…            │
│ 효과음 [＋][띠링][＋]…                    │
└──────────────────── [다시 만들기 →] ──────┘
```

- 구간 탭 → 인라인 확장: 시작/끝, 자막·훅 textarea(수동 줄바꿈 `\n` 지원), 이 구간 효과음 선택(▶), 구간 제외
- 자막 오버레이 근사: 확장 시 구간 썸네일 위에 내 자막을 실제와 같은 폰트(Pretendard)·크기 비율·위치로 겹침 — 줄바꿈 피드백 루프가 여기서 닫힘. 애니메이션 제외(데모 클립이 커버)
- 효과음 재생 시점: **구간 시작** 고정. 임의 시점 배치는 과설계로 제외

## 백엔드

- `GET /api/music`, `GET /api/sfx` — 라이브러리 목록(파일·무드/이름·크레딧). 파일 서빙은 경로 탈출 차단
- `POST /api/jobs/{id}/analysis` — segment에 `sfx` 필드(라이브러리 상대 경로 | null) 추가
- `POST /api/jobs/{id}/rebuild` — `bgm_choice` 추가: `auto`(기존: 업로드곡→무드 자동) / `off` / `"upbeat/Carefree.mp3"`(명시 선택, 업로드곡보다 우선)
- SFX 믹싱: 구간 이어붙일 때 출력 타임라인의 구간 경계에 ffmpeg 오버레이 믹싱. 숏츠엔 포함된 구간의 효과음이 따라감. 점프컷은 구간 내부만 줄이므로 구간 시작 기준 정렬이면 오차 없음
- PIL 렌더러 `wrap_text`에 수동 `\n` 우선 처리 (Remotion은 pre-wrap이라 이미 지원)

## 자산

- 스타일 데모 클립: 4종(fade/kinetic/impact/pil) × 고정 문구 2초 — 실제 렌더러로 1회 생성해 레포(`web/static/demos/`) 포함
- SFX 기본 세트: CC0 10개 내외(띠링·붐·휙·팝·박수·두둥·타다·클릭 등) + 크레딧 메타 json. 레포 `web/sfx/`에 포함(이미지에 굽힘). 확장 = mp3 추가
- BGM은 기존 hostPath(`web/music`) 유지

## 구현 순서 (각각 독립 배포 가능)

1. BGM 선택 + 전체 설정 바 + 스타일 데모 클립 + PIL `\n`
2. 타임라인 UI — 교정 패널을 트랙형으로 개편, textarea 줄바꿈, 자막 오버레이 미리보기
3. 효과음 — 라이브러리 + 구간별 배치 + 믹싱 파이프라인

## 테스트

- 라이브러리 경로 가드(탈출 차단), `bgm_choice` 해석, SFX 타이밍 계산(구간 경계→출력 시각)을 순수 함수로 분리해 pytest
- 배포 후 실기기(폰) 확인: 타임라인 가로 스크롤, 미리듣기, 공유 시트

## 제외 (YAGNI)

- 실시간 합성 미리보기, 구간별 자막 스타일/효과, SFX 임의 시점 배치, 구간 1개 프리뷰 렌더(필요해지면 다음), BGM 라이브러리 확충
